# Go-gRPC-Service

## Overview
A microservice built with Go and gRPC for high-performance inter-service communication.

## Quick Start (1-Click Build)

```bash
git clone https://github.com/skylerblue333/Go-gRPC-Service.git
cd Go-gRPC-Service
go mod tidy
go run server/main.go
```

## Features
- gRPC server implementation
- Protobuf-based message serialization
- Context-aware request handling
